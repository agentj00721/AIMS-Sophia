# AIMS Gulf × Wellx — Interactive Commercial Proposal

An interactive, storytelling-format web proposal prepared by **Wellx** for **AIMS Gulf Insurance Brokers**.

> *A single gateway to health and wellness for modern organizations.*

## About

This is a single-page, chapter-based experience (not an endless scroll). Each section of the
commercial proposal is a discrete "chapter" reached through a persistent navigation rail,
keyboard arrows (← →), on-screen controls, or the mouse wheel. On mobile it gracefully
collapses to a stacked, scroll-spy layout with a slide-out menu.

### Chapters
| # | Chapter | Content |
|---|---------|---------|
| 00 | Cover | Title, positioning, prepared-for |
| 01 | Who We Are | The three-layer platform; member / employer / insurer / broker value |
| 02 | Trusted By | Client logo wall + animated proof stats |
| 03 | The Context | The new standard for Gulf workforce wellbeing; the fragmented model |
| 04 | The Solution | One Platform · One Experience · One Outcome |
| 05 | The Platform | Gamification, HR Intelligence, Rewards, Content & Calendar |
| 06 | What HR Gains | Insights over assumptions |
| 07 | Commercial Value | Distribution models + **interactive volume-pricing calculator** |
| 08 | Portfolio & Scale | Wellx Light · X Upgrade · White-label & Ring-Fenced Cloud |
| 09 | Next Steps | Align → Pilot → Roll across the book |
| 10 | Contact | Sophia Petkova-Dubbiosi, B2B Lead |

### Interactive highlights
- **Volume pricing calculator** (Option 5) — drag the slider to model any portfolio size;
  banded tiers are computed live ("each rate applies only to members within that band").
- **Portfolio tabs** — switch between Wellx Light, X Upgrade and Scale & White-label.
- Animated proof counters, chapter reveal transitions, and a live progress rail.
